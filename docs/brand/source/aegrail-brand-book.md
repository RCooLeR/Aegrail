# Aegrail Brand Book

## Brand essence
Aegrail protects by watching evidence, not guessing. It collects logs, snapshots, diffs, file changes, admin activity, and database changes, then turns them into a defensible incident timeline.

## Taglines
- Primary: Evidence-first security audits for web incidents.
- Secondary: Follow the breach trail.

## Brand principles
1. Evidence-first: start with verifiable evidence, not assumptions.
2. Traceability: every action leaves a trace.
3. Defensible timeline: every change is chronological, ordered, and explainable.
4. Clarity: signal over noise.
5. Trust: transparent results that teams can stand behind.

## CLI language
- aegrail scan
- aegrail diff
- aegrail report
